## JavaScript

Utilities for JavaScript

### isNullOrUndefined
Checks whether a variable is null or undefined and returns `true` in that case only, other "falsy" values will return `false`.

```ts
function isNullOrUndefined(variable?: any);
```

```ts
import { JavaScript } from '@armstrong/rocketmakers';

const myString = '';
const myNumber = 0;
const myNull = null;
const myUndefined;

// this variable will be `false`
const testString = JavaScript.isNullOrUndefined(myString);

// this variable will be `false`
const testNumber = JavaScript.isNullOrUndefined(myNumber);

// this variable will be `true`
const testNull = JavaScript.isNullOrUndefined(myNull);

// this variable will be `true`
const testUndefined = JavaScript.isNullOrUndefined(myUndefined);
```
