# SCSS mixins

Utility mixins for use in SCSS. To use, simply import them from Armstrong or import your own file that imports them.

## Pseudo elements

### init-Pseudo

Adds the base properties needed to initialise a pseudo-element

```css
/* definition */
@mixin init-pseudo {
  display: block;
  content: '';
}

/* use */
@import '~@rocketmakers/armstrong-edge/dist/imports.scss';

.thing {
  &::before {
    @include init-pseudo;
  }
}
```

### range-input-thumb

Selects the thumb of a range input

```css
/* definition */
@mixin range-input-thumb {
  &::-webkit-slider-thumb {
    @content;
  }
  &::-moz-range-thumb {
    @content;
  }
  &::-ms-thumb {
    @content;
  }
}

/* use */
@import '~@rocketmakers/armstrong-edge/dist/imports.scss';

@incude range-input-thumb {
  ...some stuff;
}
```

## Layout

### fill

Makes the element fill the container

```css
/* definition */
@mixin fill {
  height: 100%;
  width: 100%;
}

/* use */
@import '~@rocketmakers/armstrong-edge/dist/imports.scss';

.thing {
  @include fill;
}
```

### absolute-fill

Makes the element fill the nearest positioned ancestor or the body

```css
/* definition */

  position: absolute;
  left: 0;
  top: 0;
  @include fill;
}

/* use */
@import '~@rocketmakers/armstrong-edge/dist/imports.scss';

.thing {
  @include absolute-fill;
}
```

### square

```css
/* use */
```

### circle

```css
  @include circle(50px);
```


## Media Queries

### screen-width-max

Wraps the content in a screen width media query

```css
/* definition */
@mixin screen-width-max($screen-width: $screen-size-phone) {
  @media screen and (max-width: $screen-width) {
    @content;
  }
}

/* use */
@import '~@rocketmakers/armstrong-edge/dist/imports.scss';

@include screen-width-max(1200px) {
  .thing {
    ...some stuff for screens smaller than 1200px;
  }
}
```
