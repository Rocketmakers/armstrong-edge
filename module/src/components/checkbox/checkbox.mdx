# Checkbox

`Checkboxes` can be used to allow the user to select multiple options.

<Canvas of={CheckboxStories.Default} layout="centered" sourceState="none" />

```tsx
<Checkbox />
```

### <a name="displaySize">`displaySize`</a>

<Canvas of={CheckboxStories.Sizes} layout="centered" sourceState="none" />

```tsx
<Checkbox displaySize="large" />
```

Size options:

- `large`
- `medium` (default)
- `small`
- `custom-<string>`

<br />

### <a name="customIndicator">`customIndicator`</a>

`Checkbox` lets you replace the tick with a custom indicator for both `checked` and `indeterminate` states.

<Canvas of={CheckboxStories.CustomIndicator} layout="centered" sourceState="none" />

```tsx
// checked indicator
<Checkbox customIndicator={<Icon />} />

// unchecked indicator
<Checkbox customIndeterminateIndicator={<Icon />} />
```
