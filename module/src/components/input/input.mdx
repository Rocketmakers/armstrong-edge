### <a name="overlay">`overlay`</a>

<Canvas of={InputStories.Overlay} layout="centered" sourceState="none" />

```tsx
<Input leftOverlay={<Icon />} />
<Input label={'Right Overlay'} rightOverlay="ml" />
```

`leftOverlay` and `rightOverlay` determine content to show on the left of the input.

<br />

### <a name="displaySize">`displaySize`</a>

<Canvas of={InputStories.Sizes} layout="centered" sourceState="none" />

```tsx
<Input displaySize="small" />
```

Size options:

- `small`
- `medium` (default)
- `large`
- `custom-<string>`
