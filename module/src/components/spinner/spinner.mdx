# Spinner

The Spinner component is commonly used to indicate loading or processing actions.

<Canvas of={SpinnerStories.Default} layout="centered" sourceState="none" />

## Spinner Props

<ArgTypes of={Spinner} />

## Default spinner

This is a basic spinner without any modifications.

```tsx
<Spinner />
```

## Custom icon spinner

The spinner accepts an icon prop that allows you to replace the default spinner.

```tsx
<Spinner icon={<Icon />} />
```

## Labelled spinner

The spinner accepts a label that can be styled separately.

```tsx
<Spinner label="loading..." />
```
