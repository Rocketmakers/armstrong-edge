# Link

### What is it?

Link is an internal Armstrong component used to give Armstrong components access to routing libraries.

For example, the `LinkButton` component replicates a `Button` but uses `Link` as its root tag instead of a

By default it renders a simple anchor tag, but this can be swapped for a Link from a routing library like `react-router`

### Use

To use the link

```tsx
import { Link } from '@rocketmakers/armstrong';

export const MyComponent: React.FC = () => {
  return (
    <>
      <Link to="/my-link">check it out</Link>

      <LinkButton<LinkProps> to="/my-link">check it out</Link>
    </>
  );
};
```

To customise the link component, use the provider at your entrypoint

```tsx
import { LinkProvider } from '@rocketmakers/armstrong';

export const App: React.FC = () => {
  return (
    <LinkProvider
      Component={({ to, children, ...props }) => (
        <a href={to} {...props}>
          {children}
        </a>
      )}
    >
      {/* the rest of the app */}
    </LinkProvider>
  );
};
```

It is important to spread the remaining props from the LinkProvider Component into your root element so that any props added when the Link is used are also spread

### React router

```tsx
import { LinkProvider } from '@rocketmakers/armstrong';
import { Link, useLocation, useHistory } from 'react-router-dom';

export const App: React.FC = () => {
  const location = useLocation();
  const { push, replace } = useHistory();

  return (
    <LinkProvider
      Component={({ to, children, ...props }) => (
        <Link to={to} {...props}>
          {children}
        </Link>
      )}
      navigate={(to, action) => (action === 'push' ? push(to) : replace(to))}
      location={location}
    >
      {/* the rest of the app */}
    </LinkProvider>
  );
};
```

Then to use the Link

```tsx

import { Link } from '@rocketmakers/armstrong';
import { LinkProps } from 'react-router';

export const MyComponent: React.FC = () => {
  return (
    <>
      <Link<LinkProps> to="/my-link">check it out</Link>

      <LinkButton<LinkProps> to="/my-link">check it out</Link>
    </>
  );
};
```

If you use the generic type on the component, it means the rest of the props on the link will be typed
