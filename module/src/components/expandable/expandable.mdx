# Expandable

An expendable component that lets you collapse or display content.

<Canvas of={ExpendableStories.Default} layout="centered" sourceState="none" />

```tsx
const [isOpen, setIsOpen] = React.useState(false);

<Button onClick={() => setIsOpen(!isOpen)}>Toggle content</Button>
<Expandable isOpen={isOpen} data-testid="expandable">
  <div>
    Some inner content
  </div>
</Expandable>
```
