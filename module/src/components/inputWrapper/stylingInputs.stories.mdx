# Styling Inputs

Almost all inputs in Armstrong use the `InputWrapper` component. This renders the div that gives them their style, and it wraps them in icons, adds a status, etc.

This means, to set the styling for your inputs, this can happen in just one place.

Here are some examples

_not all of these inputs work, this is a limitation of Storybooks Canvas not being able to have its own state, so they can't actually be easily bound up_

### Outlined

```css
.arm-input-wrapper {
  border: 1px solid $gray-light;

  &:focus-within,
  &:focus {
    border-color: $brand-primary;
  }
}
```

### Funky

```css
.arm-input-wrapper {
  background-color: $gray-dark;
  color: $white;
  border-bottom: 2px solid $brand-primary;
}

.arm-dropdown-content {
  background-color: $gray-dark;
  color: $white;
  border: 1px solid $gray-light;
}
```
