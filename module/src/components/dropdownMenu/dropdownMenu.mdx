# Dropdown menu

A dropdown menu featuring section dividers and icon overlays.

<Canvas of={DropdownMenuStories.Default} layout="centered" sourceState="none" />

```tsx
import { DropdownMenu, Button } from '@rocketmakers/armstrong';

const items = [
  {
    label: 'Item',
    onClick: () => {},
  },
]

<DropdownMenu items={items}>
  <Button>Toggle item list menu</Button>
</DropdownMenu>
```

### <a name="items">Dropdown menu items</a>

### Basic item

```tsx
<DropdownMenu items={[{ label: 'Item', onClick: () => {} }]}>
  <Button>Button</Button>
</DropdownMenu>
```

### Non-clickable item

```tsx
<DropdownMenu items={[{ label: 'Non-clickable item' }]}>
  <Button>Button</Button>
</DropdownMenu>
```

### Disabled item

```tsx
<DropdownMenu items={[{ label: 'Disabled item', disabled: true }]}>
  <Button>Button</Button>
</DropdownMenu>
```

### Item with icon overlay

```tsx
<DropdownMenu
  items={[
    {
      label: 'Icon on the left',
      leftOverlay: <Icon />,
    },
    {
      label: 'Icon on the right',
      rightOverlay: <Icon />,
    },
  ]}
>
  <Button>Button</Button>
</DropdownMenu>
```

<br />

### <a name="header-footer">Dropdown menu header and footer</a>

### Header

```tsx
<DropdownMenu items={items} headerContent={<div>Header content</div>}>
  <Button>Button</Button>
</DropdownMenu>
```

### Footer

```tsx
<DropdownMenu items={items} footerContent={<div>Footer content</div>}>
  <Button>Button</Button>
</DropdownMenu>
```

<br />

### <a name="arrow">Dropdown menu with arrow</a>

```tsx
<DropdownMenu items={items} showArrow>
  <Button>Button</Button>
</DropdownMenu>
```
### Non-modal dropdown menu

Stops the dropdown menu blocking interaction with the rest of the page when open.

```tsx
<DropdownMenu items={items} modal={false}>
  <Button>Button</Button>
</DropdownMenu>
```
