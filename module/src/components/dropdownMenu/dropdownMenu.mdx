# Dropdown menu

A dropdown menu featuring section dividers and icon overlays.

<Canvas of={DropdownMenuStories.Default} layout="centered" sourceState="none" />

## Dropdown menu items

### Basic item

```tsx
<DropdownMenu items={[{ label: 'Item', onClick: () => {} }]}>
  <Button>Button</Button>
</DropdownMenu>
```

### Non-clickable item

```tsx
<DropdownMenu items={[{ label: 'Non-clickable item' }]}>
  <Button>Button</Button>
</DropdownMenu>
```

### Disabled item

```tsx
<DropdownMenu items={[{ label: 'Disabled item', disabled: true }]}>
  <Button>Button</Button>
</DropdownMenu>
```

### Item with icon overlay

```tsx
<DropdownMenu
  items={[
    {
      label: 'Icon on the left',
      leftOverlay: <Icon />,
    },
    {
      label: 'Icon on the right',
      rightOverlay: <Icon />,
    },
  ]}
>
  <Button>Button</Button>
</DropdownMenu>
```

## Dropdown menu with header and footer

### Header

```tsx
<DropdownMenu items={items} headerContent={<div>Header content</div>}>
  <Button>Button</Button>
</DropdownMenu>
```

### Footer

```tsx
<DropdownMenu items={items} footerContent={<div>Footer content</div>}>
  <Button>Button</Button>
</DropdownMenu>
```

## Dropdown menu with arrow

```tsx
<DropdownMenu items={items} showArrow>
  <Button>Button</Button>
</DropdownMenu>
```
