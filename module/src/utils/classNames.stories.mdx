## ClassNames

### Concat

Based on [classNames](https://www.npmjs.com/package/classnames) but included in Armstrong to reduce dependencies

```ts
import { ClassNames } from '@rocketmakers/armstrong';

const className = ClassNames.concat('thing', 'other-thing', { ['another-thing']: true, ['an-unused-thing']: false }, ['hello', 'there']);

// className
('thing other-thing another-thing hello there');
```
