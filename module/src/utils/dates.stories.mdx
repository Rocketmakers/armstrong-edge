## Dates

Utilities for working with dates. Armstrong uses date-fns under the hood.

### dateLikeToDate

Takes a `DateLike` (Date|string|number) object and parses it to a date `Date` object

Uses the supplied format string and locale if date is a string.

```ts
function dateLikeToDate(date: DateLike | undefined, formatString?: string, locale: Locale = defaultLocale): Date | undefined;
```

```ts
import { Dates } from '@rocketmakers/armstrong';

const dateString = '29-09-2021';

const date = Dates.dateLikeToDate(dateString, 'dd-mm-yyyy');
```

### dateToString

Takes a native `Date` object and returns a formatted string.

Uses the supplied format string and locale to format the string.

```ts
function dateToString(date: Date, formatString?: string, locale: Locale = defaultLocale): string;
```

```ts
import { Dates } from '@rocketmakers/armstrong';

const result = Dates.dateToString(new Date(), 'dd-mm-yyyy');
```

### dateObjectToDateLike

Turns a date object into a `DateLike` matching the requested type.

```ts
function dateObjectToDateLike(date: Date, type: string, formatString?: string, locale: Dates.DateLocale = Dates.defaultLocale): Dates.DateLike;
```

```ts
import { Dates } from '@rocketmakers/armstrong';

const result = Dates.dateObjectToDateLike(new Date(), 'string');
```
